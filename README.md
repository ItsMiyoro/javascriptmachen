# 10 przykazań JS
## Jam jest kod twój który wywiódł cię z ziemi assembly z domu fortrana
1. Nie będziesz miał kodów closed source przed mną
2. Nie będziesz brał imienia java skrypta swojego nadaremno
3. Pamiętaj abyś kod święty święcił
4. Czcij java skrypta i type skrypta swego
5. Nie sudo rm -rf / --no-preserve-root
6. Nie cudzołóż innych kodów
7. Nie kradnij innych kodów
8. Nie używaj fałszywego AI przeciw kodu swemu
9. Nie pożądaj kodu swojego bliźniego
10. Ani żadnego commita który jego jest
